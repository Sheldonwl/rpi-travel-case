# Hardware
![Alt text](/docs/images/pinout.png?raw=true "Raspberry Pi 4B Pinout")

## Dual fan heatsink 
**Attach the following pins** 
4 - 5v Power
6 - Ground 

## Blinkt 
**Attach the following pins** 
2 - 5v Power
14 - Ground 
16 - Data
18 - Clock 

## Button
**Attach the following pins** 
3 - Output pin 
20 - Ground
